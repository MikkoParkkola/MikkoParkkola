# Open-Source Tools

```bash
brew tap MikkoParkkola/tap
```

| Tool | Install | What |
|---|---|---|
| **trvl** | `brew install MikkoParkkola/tap/trvl` | Google Flights + Hotels MCP server. 32 tools, trip optimization, travel hacks. Free, no API keys. |
| **axterminator** | `brew install MikkoParkkola/tap/axterminator` | macOS GUI automation MCP server. 30 tools, background testing, audio capture, virtual desktops. |
| **mcp-gateway** | `brew install MikkoParkkola/tap/mcp-gateway` | Universal MCP gateway. Single port for all your MCP servers, ~95% context token savings. |
| **nab** | `brew install MikkoParkkola/tap/nab` | URL to clean markdown for LLMs. HTTP/3, JS rendering, cookie auth, anti-fingerprinting. |
| **nowifi** | `brew install MikkoParkkola/tap/nowifi` | Captive portal bypass. One command, 27 techniques, auto-restores on Ctrl+C. |

## Wire into Claude Desktop

```bash
brew install MikkoParkkola/tap/trvl
trvl mcp install-claude-desktop
# restart Claude Desktop — 32 travel tools ready
```

Same pattern works for `axterminator` and `mcp-gateway`.
