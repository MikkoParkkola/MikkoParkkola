# Contributing

Thanks for helping improve this profile repository.

## Scope

This repo is a landing page for public projects under `MikkoParkkola`.

- **Changes that belong here:** README fixes, broken links, install instructions, repo descriptions, and discoverability improvements
- **Changes that do not belong here:** bugs, feature requests, or security reports for `trvl`, `axterminator`, `mcp-gateway`, `nab`, `nowifi`, or other linked projects

For project-specific issues, use the canonical repository for that tool.

## Workflow

1. Fork the repository
2. Create a focused branch
3. Make a small, reviewable documentation change
4. Verify links, commands, and formatting before opening a pull request

## Pull Requests

- Keep changes limited to the profile content or repository metadata
- Update install commands and links together if the project list changes
- Include enough context for reviewers to understand why the landing page needs the update
